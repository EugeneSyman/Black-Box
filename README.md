## Black Box

An application that allows you to record sound in normal and background mode, create schedules for turning on sound recording in the background, also supports backup to cloud storage,
Encryption of the database and activation of sound recording using a key combination

Additional functionality is available to the user after clicking on the button in the upper right corner "Main". After clicking on it, all subsequent user clicks on the sound buttons ( + , - ) will be processed by the application itself. When you hold the key up, all the way (3 seconds), recording will begin. Which can be stopped by the reverse action, hold down the button. This combination to activate recording in the background also works when the screen is locked
